Part 1 — Stand up a project

Clone forge → create a new project (idea seeded, contract items placed, KB seeded to grow) or onboard an existing repo (align to the contract), with AI-assisted generation (architect, instructions, project brain).

Part 2 — Compose the four pillars

Flows, Skills, Agents, Knowledge — each out-of-the-box, editable, and user-creatable. The pipeline is data you build, not a script.

Part 3 — Run a gated cycle

Idea → PLAN gate → autonomous build → verdict gate (progress the PR through the flow UI) → merge → reflect. The out-of-the-box flow makes a real, reviewed addition.
