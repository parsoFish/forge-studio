# Archived documentation

Point-in-time docs preserved for history — **NOT current state**. For the current
architecture see [`docs/architecture/refocus-architecture/`](../architecture/refocus-architecture/README.md);
decision records live in [`docs/decisions/`](../decisions/) (ADRs); active phase docs in
[`docs/phases/`](../phases/).

- **planning/** — dated planning / decision-session artifacts (the 2026-05-* refinement,
  brain-refinement, rebuild-review, thin-forge, brain-audit, and claude-trail-rebuild sessions).
- **autonomous-runs/** — operator run reports; the gaps they surfaced are folded into
  [`docs/known-gaps.md`](../known-gaps.md).
- **architecture/** — the 2026-05-17 as-built snapshot, the legacy `forge2.0` swimlane diagram,
  and the C4 model — all superseded by `refocus-architecture/` as the canonical current architecture.
- **verifications/** — UI verification-run evidence from past cycles.
- **seeding-plan.md**, **forge-user-stories.md** — completed prospective plan / traceability records.
